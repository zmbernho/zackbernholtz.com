UNIVERSITY OF WATERLOO ALTERNATIVE FUELS TEAM

In collaboration with the University of Waterloo Alternative Fuel's Team I worked with a friend to do the sensor characterization and create the algorithm for a automatic backup object detection system. The goal of the project was to integrate and characterize the rear facing ultrasonic sensors. After completing testing of the sensors we installed and wired the sensors to test their communication on the vehicle's CAN bus. We performed further testing with this setup in the vehicle to allow for the driver to have rear facing sensors detect objects only duirng parking maneuvers.

A large part of the project involved sensor characterization and CAN bus debugging of the vehicle. I was able to gain sensor fusion knowledge as well, as I wrote the algorithm to combine multiple sensor data from the four ultrasonic sensors to create an occupancy grid.
